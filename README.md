## Who I Am

I'm a **Full-Stack Software Engineering** student at **Holberton School** in Albania, building a strong foundation across the full spectrum of software — from low-level systems programming in C to web applications and machine learning.

I believe in learning by building. I don't just consume knowledge — I apply it, break it, and build it again until it makes sense.

Right now I'm diving deep into **Machine Learning** and **Neural Networks**, exploring how intelligent systems learn from data. The intersection of mathematics, code, and intuition is what keeps me hooked.
